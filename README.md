<h1>Packing</h1>
<h3>Общая информация</h3>
<span>Язык программирования: C++</span><br>
<span>Графическая библиотека: SFML</span><br>
<span>Год: декабрь 2020</span><br>
<h3>Описание</h3>
<p>Программа создана с целью дать пользователю понять, можно ли разместить на плоскости определенной площади объекты с известной площадью и как это сделать максимально эффективно. Управление представлено ниже:
</p>
<ul>
<li>Создание фигурки на поле MOUSE_RIGHT_CLICK</li>
<li>Радиус фигурки: 1 - увеличение, 2 - уменьшение</li>
<li>Порог: ARROW_UP - увеличение, ARROW_DOWN - уменьшение</li>
</ul>
